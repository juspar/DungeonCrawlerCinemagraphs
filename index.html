Intent & Purpose

This application is an unofficial, fan-made companion display designed to provide atmospheric, infinitely looping ambient art ("cinemagraphs") for readers of the Dungeon Crawler Carl series by Matt Dinniman. The visuals are designed to set the mood of each floor/chapter without drawing active attention away from the text.

How to Use

  • Single Scene: Click on any unlocked slot in the grid to launch the scene in full-screen ambient mode.
  • Random Cycling: Click Enter Select Mode at the bottom-left of the screen, select multiple scenes by clicking slots in the grid, choose a transition duration, and click Start Playlist.
  • Ambient Controls: Use the exit button in the top-left to return, or use the navigation arrows to switch scenes.

Development Details

Each scene is constructed using multiple transparent WebP layers positioned in a 3D perspective camera space. Idle animations are driven entirely by hardware-accelerated CSS keyframes, while dynamic environmental elements (like steam, stars, sparkling sand, dust, and bubbles) are simulated in real-time using custom HTML5 Canvas physics engines for maximum efficiency during long reading sessions.
